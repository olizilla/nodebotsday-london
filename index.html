What is it?

NodeBots of London is a semi regular meetup to hack on hardware using JavaScript.

We'll build robots, fly node copters and perhaps inadvertently kickstart the robot apocalypse.

What should I bring?

All you need is a laptop. The NodeBots of London Esoteric Hardware Library stands ready to provide for your hardware needs on the day but if you are a beginner you may wish to pick up an Arduino Starter Kit to continue hacking at home.

Otherwise if you have a project, bring it along to get help and feedback on your idea.

Soldering irons, dremels and other tools are available for you to use.

Do I need to do anything before I arrive?

If you are a beginner, try the nodebot-workshop - it will give you a feel for hardware programming without needing access to any actual hardware.

Please ensure you have a working C compiler installed - on Windows install Visual Studio, on a Mac please install Xcode. On Linux you probably have one already.
