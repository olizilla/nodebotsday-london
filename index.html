NodeBots Day London?

On July 27th, 2013, we'll meet up in London and hack on Arduinos, Rasberry Pis, NodeCopters, robots, and any other gizmos that need more input...

NodeBots Day?

On July 27th, 2013, people all over the world will meet up to build robots, program copters, solder, hack and learn. Events are already schdueled for: San Francisco, New York, Sydney, and more...

NodeBots?

Robots, powered by node.js. That's right, if you've any JavaScript experience, you can now use your powers to manipulate the machines. From blinking lights through to Sumo Bot battles, Remote control cats, and Bull fighting with helicopters.

Need more input?

For this event, you will need to bring your own hardware and ideas. Don't be scared. That means a laptop, an Arduino, curiosity and components available from all good widget retailers

If you're just gettting started then a starter kit might help:

  • Hobbytronics: SparkFun Inventor's Kit for Arduino
  • Hobbytronics: Zumo Robot Kit
  • CPC: Arduino.cc Starter kit
  • oomlout: Starter Kit for Ardunio

Don't worry if you don't have all the things, there will be a nodecopter or 2, a rasberry pi, and other people with similar problems to solve.

We'll be using @rwaldron's johnny-five module. The gitub repo has the docs and links to great tutorials and there are more tutorials on the bocoup site. If you want to get a headstart, then you should install node.js and johnny-five, plug in to your Ardunio, and start strobing LED's and moving matter.
